This_DeMo is a portable Spring Boot application built on Java 1.7 and clearly hosted on AWS.

Its controllers return MVC templates in addition to static hosting and a somewhat promising RESTful service.

The application is Maven-packaged as a single JAR executable file (WAR available on request).

Contained herein are a configurable Tomcat server, an underutilized pSQL database, and a seasoned understanding of this web stack.

Each client-side UI template is rendered responsively and should compile on any device.

The content and effects are hand-coded from CSS and HTML code on a Skeleton frame.
